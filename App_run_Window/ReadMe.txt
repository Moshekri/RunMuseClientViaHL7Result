
This Application will start a muse editor directly with a specific ecg record loaded.

Usage:

you can copy & paste a url or a part of the url or the entire HL7 content in the url source window.
note : the text must contain at leat this data from the url for this to work: 

?PatientID=064534423\T\Date=06-03-2014\T\Time=01%3a55%3a18%3a00\T\TestType=ECG\T\Site=1
